# BSPWM en KALI

  Configuracion de BSPWN, Polybar, sxhkd, rofi, nvim, picom, alacritty, mas.
 
## Info
```bash
Theme         = None
WM            = BSPWM
Bar           = Polybar
Terminal      = alacritty
Font          = Victor Mono Light
Wallpapers    = variety
Shell         = oh-my-zsh or oh-my-fish
Editor        = nvim
Notifications = rofi
``` 
## Instalation

1) Clonar el Repositorio:
  
  ``` bash
  git clone https://github.com/AlonsoErnesto/Dotfiles2023.git
  cd Dotfiles2023
  cp * ${HOME}/.config/
  ```
  
2) Finish!
